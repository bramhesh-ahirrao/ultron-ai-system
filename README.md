# 🔴 Ultron AI System

A futuristic AI terminal interface powered by local LLM using Ollama.

---

## 🚀 Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: Flask (Python)  
- AI Engine: Ollama (Llama 3)  
- Database: MongoDB  
- AI Feature: RAG (PDF-based document understanding)

---

## ⚡ Features

- 🧠 AI-powered terminal interface  
- 💬 Command system (reboot, purge, status)  
- 🔗 Flask API integration  
- 🗄️ MongoDB chat memory  
- 📄 PDF-based AI (RAG system)  
- ⚡ Real-time interaction
